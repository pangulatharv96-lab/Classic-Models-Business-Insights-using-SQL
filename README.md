# Classic-Models-Business-Insights-using-SQL
Project Overview
Classic Models Business Insights using SQL
(Sales & Customer Analysis of Classic Models Database)
This project focuses on analyzing the Classic Models relational database to extract meaningful business insights using SQL. The dataset represents a real-world company that sells scale model vehicles and includes information about customers, orders, products, employees, offices, and payments.
The objective of this project is to understand sales performance, customer behavior, product demand, and revenue trends by writing optimized SQL queries and performing exploratory data analysis. Various business questions such as top customers, yearly revenue, popular product lines, stock availability, and employeeâ€“customer relationships are answered through structured queries.

Tools & Technologies
Database: MySQL
Language: SQL

Key Outcomes
Revenue analysis by year
Identification of top customers and high-value payments
Product and inventory analysis
Customer order and payment behavior analysis
Employee and office-level insights
